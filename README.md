## 開發工具
Node.js @ 14.18.1

## 執行方式
1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
```
git clone
```
3. 安裝 npm 套件
```
git install
```
4. 啟動專案
```
npm start
React 將自動開啟瀏覽器渲染畫面 (網址 :  http://localhost:3000)
```
5. 若欲暫停使用
```
ctrl + c
```
